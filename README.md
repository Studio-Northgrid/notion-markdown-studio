# notion-markdown-studio
---
Ein schlankes Browser-Tool um Notion-Exporte sauber als PDF aufzubereiten, keine Installation, kein Server, läuft komplett lokal im Browser
---
## Features
- Live A4-Vorschau
- Manuelle Seitenumbrüche per Button oder <!-- pagebreak -->
- Syntax Highlight für Code-Blöcke
- Farbige Überschriften im Editor und der Vorschau
- PDF-Export über Druck Dialog, Farben und Formatierungen bleiben erhalten
---
## Verwendung
1. Notion-Seite exportieren als Markdown & CSV
2. Die .md Datei oben per Button öffnen oder direkt kopieren
3. PDF/Druck --> als PDF speichern
---
## Warum
Notions eigener PDF-Export ist unzuverlässig. Seitenumbrüche mitten im Block, Code ohne Highlight, kaum Kontrolle über das Layout. Dieses Tool gibt dir diese Kontrolle zurück, ohne das du etwas installieren musst.
